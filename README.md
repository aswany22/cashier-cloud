# CashierCloud

هذا هو خادم الربط السحابي الحقيقي بين برنامج الكاشير وتطبيق متابعة المبيعات.

## المهم

تشغيل الخادم على الكمبيوتر المحلي لا يحقق المتابعة من أي مكان. هذا يصلح للتجربة فقط.

للمتابعة من أي شبكة، يجب نشر هذا المشروع على استضافة إنترنت برابط ثابت مثل:

```text
https://api.your-domain.com
```

بعدها تضع هذا الرابط داخل تطبيق الموبايل.

## ملفات النشر

- `Dockerfile`: لتشغيل الخادم على VPS أو أي منصة تدعم Docker.
- `publish/`: نسخة نشر عادية بدون Docker.
- `data/cashier-cloud-db.json`: قاعدة البيانات الملفية.

## متغيرات مهمة

غيّر رمز API قبل النشر:

```text
ApiToken
```

في الاستضافة اجعله قيمة قوية مثل:

```text
ApiToken=YOUR_SECRET_TOKEN
```

## endpoints

- `GET /health`
- `POST /api/sync/push`
- `GET /api/sync/pull`
- `GET /api/reports/summary`

كل مسارات `/api/*` تحتاج:

```text
X-Api-Key: YOUR_SECRET_TOKEN
```

## طريقة الربط بعد النشر

داخل تطبيق الموبايل:

- رابط السيرفر: `https://api.your-domain.com`
- رمز API: الرمز الذي وضعته في الاستضافة

ثم:

- `تنزيل من السحابة`: لمتابعة المبيعات والجرد.
- `رفع للسحابة`: لإرسال أي منتجات أضيفت من الموبايل.

## المطلوب لإكمال الربط فعليًا

واحد من هذه الخيارات:

1. بيانات VPS أو استضافة تدعم Docker.
2. حساب منصة نشر مثل Azure أو Render أو Railway أو Fly.io.
3. دومين اختياري، أو رابط مباشر من منصة الاستضافة.

بدون استضافة عامة، التطبيق لن يعمل من أي شبكة.
